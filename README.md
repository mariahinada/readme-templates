# readme-templates

# Descrição do Repositório

Este repositório contém modelos de README para ajudar os desenvolvedores e QA's a documentarem seus projetos.

# Contribuições

Se quiser contribuir, clone este repositório, crie sua própria branch e envie o seu PR para revisão.

Você pode:
- Criar novos modelos
- Melhorar templates existentes
- Corrigir erros ou ajustar formatação
- Adicionar versões alternativas (frontend, backend, QA, etc.)

1. Clone o repositório
```
git clone https://github.com/mariah-inada/readme-templates.git
```
2. Crie uma nova branch
```
git checkout -b feature/NOME
```
3. Faça as suas alterações e envie seu PR.
